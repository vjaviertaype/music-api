Music API

Accedé a artistas, álbumes y canciones con Music API.

¿Qué es la Music API?

La Music API te permite buscar artistas, ver sus álbumes y canciones, y suscribirte para recibir notificaciones sobre nuevos lanzamientos.

Entidades de la API

La API está estructurada en entidades relacionadas con la industria musical y la gestión de usuarios.

Usuario

Representa a una persona registrada en el sistema, identificada mediante un correo electrónico y utilizada para gestionar preferencias y suscripciones.

Artista

Representa a un creador musical y contiene su información básica.

Álbum

Agrupa un conjunto de canciones publicadas por un artista en una fecha determinada.

Canción

Describe una obra musical individual, incluyendo su título, duración y metadatos asociados.
